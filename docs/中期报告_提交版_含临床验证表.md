# 医学部大学生创新实验项目中期报告（提交版）

## 中期报告正文（1000字内）
本项目围绕“面向外科手术的双模式（随动/姿态保持）上肢辅助外骨骼”开展中期研发，目标是建立可用于术中减负与稳定支撑的人机协作系统。阶段内已完成三方面工作：一是结构与系统方案，完成核心部件三维建模、装配关系校核及关键机构样机验证；二是控制与软件，完成模式切换状态机与脚踏确认链路设计，形成“候选意图—人工确认—执行切换”的共享控制逻辑；三是数据与算法，完成基于JIGSAWS（Knot_Tying、Needle_Passing、Suturing）的手术意态感知程序，建立从数据清洗、滑窗特征提取到训练推理与可视化报告的完整流程。当前离线测试结果为：103条试验数据、7957个窗口，测试集准确率0.602，Macro-F1为0.578；系统给出“建议锁定”判据为P(lock)≥0.58且global_vel_mean≤0.063，能够较稳定区分“精细稳态阶段（建议锁定）”与“重定位/大运动阶段（不建议锁定）”。

项目实施累计投入约430人时：文献调研与临床需求梳理约80人时，机械设计与迭代约130人时，控制软件开发约140人时，数据标注清洗与模型训练约80人时。项目资助经费为4500元，前期已按研发需求启动部分支出，主要用于样机加工与结构件制作、连接件与固定绑带等装配材料、传感器接口与实验辅材购置，以及答辩展示与图文材料输出；其余经费将继续用于临床前验证阶段的实验耗材、受试者测试准备和记录分析材料。阶段性成果包括：完成MVP级手术意态感知程序与展示图表；形成临床验证实验方案与标准化数据记录表；申请2项相关专利。当前主要问题有三点：其一，公开数据库与真实术中数据存在域差异，模型跨场景泛化仍需提升；其二，临床受试数据尚在伦理与招募准备阶段，真实闭环证据不足；其三，硬件传感与模式切换时序同步仍需进一步优化。下一阶段将优先完成伦理流程与小样本临床前验证，接入真实IMU与外骨骼本体传感器，开展“无外骨骼/随动/锁定”对照实验，重点验证肌电负荷、轨迹精度、疲劳发展与可用性，形成可复现的统计结论。通过以上工作，项目已从概念设计进入“可运行、可评估、可转化”的关键阶段。
